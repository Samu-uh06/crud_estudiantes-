# Sistema de Gestión de Estudiantes - CRUD en Dart

## Descripción
Aplicación CRUD desde consola para gestionar estudiantes de un curso.

## Requisitos
- Dart SDK instalado

## Ejecución
```bash
dart main.dart
```

## Funcionalidades
1. **Agregar estudiante**: Registra nombre, edad y 3 notas (calcula promedio automáticamente)
2. **Listar estudiantes**: Muestra todos los estudiantes con sus datos
3. **Actualizar estudiante**: Modifica datos por índice (Enter mantiene valor anterior)
4. **Eliminar estudiante**: Elimina por índice
5. **Promedio general**: Calcula estadísticas del curso
6. **Salir**: Finaliza el programa

## Validaciones
- Notas entre 0 y 5
- Edad mayor a 0
- Índices válidos
- Entradas no vacías
- Datos numéricos correctos
